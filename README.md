# nft-marketplace
This is a WIP

## 
```
Solidity
Web3
ERC721
Truffle
Open
React JS
Redux

```

## Setup

```
Clone the repository
Create a .env file and populate the enviroment variables
Download and install Ganache
run npm run install
```

## Run
```
npm run dev
```

## Enviroment variables
```
REACT_APP_API_URL
RPC_URL_BSC
RPC_URL_RINKEBY
RPC_URL_BSC_TESTNET
RPC_URL_GANACHE
NFT_CONTRACT_ADDRESS
MARKETPLACE_CONTRACT_ADDRESS
```
