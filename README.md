Solidity

## Setup

```
Clone the repository
Create a .env file and populate the enviroment variables
Download and install Ganache
run npm run install
```

## Run
```
npm run dev
```

## Enviroment variables
```
REACT_APP_API_URL
RPC_URL_BSC
RPC_URL_RINKEBY
RPC_URL_BSC_TESTNET
```
